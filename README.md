# JoppOS
### What is JoppOS
JoppOS is a x86_64 operating system for all occasions.
### Advantages of JoppOS
The most obvious advantage is that JoppOS's code is in the public domain.
JoppOS also comes with many user tools out of the box. For example, it already has GCC, JPASM, GIMP, and Firefox.
JoppOS also has a very user-friendly and fast GUI.
